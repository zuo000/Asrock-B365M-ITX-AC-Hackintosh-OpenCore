## 硬件配置
我这台黑苹果主机的整体配置清单如下：

- `cpu`：Intel i5 9400F
- `主板`：华擎 B365M-ITX/AC
- `显卡`：蓝宝石 RX560XT 8G
- `硬盘`：光威 悍将 2.5寸 1.5T SSD 
- `内存条`：威刚XPG 8G*2 DDR4
- `散热器`：金河田 9cm四铜管风扇
- `电源`：全汉 450W
- `机箱`：酷鱼 G5 Plus
- `无线网卡`：博通 BCM943602CS

## OpenCore
版本0.6.3，支持GUI界面，图标是我精心挑选组合的一套，不喜欢的可以自行更换，勿喷^_^
